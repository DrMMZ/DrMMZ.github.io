2.2 Train/Dev/Test

For later training purpose, we normalize the data.

normalizedD = D

intL = c()
for (x in names(D)) {
  if (class(D[,x])=="integer" | class(D[,x])=="numeric") {
    normalizedD[,x] = (D[,x] - mean(D[,x])) / sd(D[,x])
    intL = c(intL, x, mean(D[,x]), sd(D[,x])) 
  }
}

mat_intL = matrix(intL, nrow = 10, ncol = 3, byrow = T)
colnames(mat_intL) = c("feature", "mean", "sd")

df_intL$mean = as.numeric(df_intL$mean)

3. Modeling

We will use models: decision tree, logistic regression and neural networks.

For later training purpose, we combine train and dev set into one train set.

rminerD1 = rbind.data.frame(XY_train, XY_dev)
rminerD2 = rbind.data.frame(nXY_train, nXY_dev)

3.1 Decision Tree

The decision tree will be applied by the package rminer of R. The model uses the holdout method on the combined train/dev (rminerD1) set with further splitting 0.8/0.2 ratio, repeated 20 times.

dt1 = mining(y~., rminerD1, method=c("holdout", 0.8), model="dt", Runs=20)
res_dt1 = mmetric(dt1, metric = c("F1", "AUC"))
df_dt1 = data.frame("F1"=mean(res_dt1[,2])/100, "AUC"=mean(res_dt1[,3]))
row.names(df_dt1) = "Decision Tree"
df_dt1

3.2 Logistic Regression

The logistic regression will be applied by the package rminer of R. The model uses the holdout method on the combined train/dev (rminerD2) set with further splitting 0.8/0.2 ratio, repeated 20 times.

lr1 = mining(y~., rminerD2, method=c("holdout", 0.8), model="lr", Runs=20)
res_lr1 = mmetric(lr1, metric = c("F1", "AUC"))
df_lr1 = data.frame("F1"=mean(res_lr1[,2])/100, "AUC"=mean(res_lr1[,3]))
row.names(df_lr1) = "Logistic Regression"
df_lr1

3.3 2-Layer Neural Network

The neural network will be applied by the package rminer of R. In particular, the model is a single hidden layer (or 2-layer) neural network with 6 units and 7 ensembles used BFGS Quasi-Newton optimizer. The model uses the holdout method on the combined train/dev (rminerD2) set with further splitting 0.8/0.2 ratio, repeated 20 times.

nn1 = mining(y~., rminerD2, method=c("holdout", 0.8), model="mlp", size = 6, nr = 7, maxit = 500, Runs=20)
res_nn1 = mmetric(nn1, metric = c("F1", "AUC"))
df_nn1 = data.frame("F1"=mean(res_nn1[,2])/100, "AUC"=mean(res_nn1[,3]))
row.names(df_nn1) = "2-Layer Neural Network"
df_nn1

3.4 3-Layer Neural Network

The neural network will be applied by the package nn_model_np implemented by the author in Python. In particular, the model (after tuning) is an 2 hidden layer (or 3-layer) neural network with 5 and 2 units, respectively, with relu-relu-sigmoid activation functions used gradient descent with momentum optimizer (0.3 learning rate, 0.9 moving average rate, 32 mini-batch size and 500 epochs). The model is trained on the train set (nXY_train).

Before training, we need to convert the data in the format used in nn_model_np of Python. For training in Python, code can be found in here.
